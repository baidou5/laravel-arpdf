# إعلان الإصدار v2.0.0 - Laravel ArPDF

تم إطلاق `laravel-arpdf v2.0.0` بتاريخ **6 مارس 2026**.

أهم تغيير في هذا الإصدار:
- إعادة بناء المكتبة بالكامل لتصبح **Driver-Based** وقابلة للتوسعة للمشاريع الكبيرة.
- إزالة الاعتماد على `mPDF` بشكل نهائي.
- اعتماد `Dompdf` كمحرك افتراضي.

ما الذي ستحصل عليه الآن:
- معمارية أنظف وأسهل للصيانة.
- سلوك أكثر أمانًا مع بيئات workers طويلة العمر.
- إمكانية ربط محرك PDF مخصص عبر `PdfEngine`.
- الحفاظ على أغلب الـ API الحالية مثل `loadHTML`, `loadView`, `download`, `stream`.

مهم قبل الترقية:
- راجع ملف `UPGRADE.md` لأنه يوجد تغييرات كاسرة للتوافق (`Breaking Changes`).

شكراً لكل من يستخدم المكتبة ويساهم في تطويرها.
